# Customer Retention & Revenue Optimization

## 📌 Objective
Analyze customer churn and provide data-driven product recommendations to improve retention and revenue.

## 🛠 Tools Used
- SQL (MySQL)
- Excel

## 📊 Key Insights
- High churn (42%+) observed in month-to-month contracts
- Long-term contracts significantly reduce churn (~2.8%)
- New customers (low tenure) show highest churn risk

## 💡 Product Recommendations
- Improve onboarding experience for new users (first 30–60 days)
- Promote long-term subscriptions with discounts and incentives
- Target high-risk users with personalized retention strategies

## 🚀 Business Impact
This analysis helps identify revenue loss drivers and provides actionable strategies to improve customer retention and long-term growth.
